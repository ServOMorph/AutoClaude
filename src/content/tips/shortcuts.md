## App AutoClaude

| Touche / Action | Effet |
|-----------------|-------|
| ESC | Arrêter autoclick |
| Clic overlay | Toggle ON/OFF |
| Fermer fenêtre | Arrêt propre + sauvegarde |

## Claude Code (IA)

| Commande | Action |
|----------|--------|
| `/start` | Démarrer session + charger apprentissages |
| `/close` | Clôturer session + documenter + commit |
| `/doc` | Audit cohérence doc (v2.6.0+) |
| `/bump_version` | Bump version automatique |

## Workflow multi-IA

- Charger les 4 organes : README, ROADMAP, ARCHITECTURE, WORKFLOW
- Budget token : ~5000-6500 tokens par session
- Handoff : créer `HANDOFF_<id>.md` si contexte > 60%
